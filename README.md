# pet_shop

🐾 Pet Love - Sistema de Gestão para Pet Shop
📋 Sobre o Projeto

O Pet Love é um sistema de gerenciamento para pet shops desenvolvido para facilitar o controle de clientes, pets, agendamentos e serviços. O objetivo é oferecer uma plataforma simples, intuitiva e eficiente para melhorar o atendimento e a organização do estabelecimento.

🚀 Funcionalidades
🐶 Cadastro de Pets
👤 Cadastro de Clientes
📅 Agendamento de Banho e Tosa
💉 Controle de Vacinação
🛒 Gerenciamento de Produtos
📊 Relatórios e Estatísticas
🔍 Pesquisa de Clientes e Pets
📱 Interface Responsiva

⚙️ Instalação
1. Clone o repositório
git clone https://github.com/seu-usuario/pet-love.git
2. Entre na pasta do projeto
cd pet-love
3. Instale as dependências
npm install
4. Configure o banco de dados

Crie um banco MySQL e configure as credenciais no arquivo:

DB_HOST=localhost
DB_USER=root
DB_PASSWORD=sua_senha
DB_NAME=petlove
5. Execute o projeto
npm start
